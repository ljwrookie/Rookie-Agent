# Examples

End-to-end examples demonstrating Rookie Agent capabilities.

## Available Examples

- [Fix Issue](/examples/fix-issue) — Debug and fix a production bug
- [Codebase QA](/examples/codebase-qa) — Answer questions about your codebase
- [PR Review](/examples/pr-review) — Automated pull request review
- [Daily Standup](/examples/daily-standup) — Generate standup updates from commits

## Running Examples

Each example includes:
1. Task description
2. Expected commands
3. Sample output

```bash
rookie init --task "Fix the memory leak in user service"
rookie verify --feature memory-leak-fix
```
