# CLI Commands

## Core Commands

| Command | Description |
|---------|-------------|
| `rookie init` | Initialize a Harness session |
| `rookie resume` | Resume a session |
| `rookie progress` | Show progress |
| `rookie verify` | Run feature verification |
| `rookie code` | Start coding assistant |
| `rookie chat` | Interactive chat |

## Management Commands

| Command | Description |
|---------|-------------|
| `rookie config` | Show merged settings |
| `rookie memory` | Show user model |
| `rookie doctor` | Health check |
| `rookie update` | Self-update |

## Hook Commands

| Command | Description |
|---------|-------------|
| `rookie hook list` | List hooks |
| `rookie hook add` | Add hook |
| `rookie hook test` | Test hook |
| `rookie hook remove` | Remove hook |

## Permission Commands

| Command | Description |
|---------|-------------|
| `rookie permission list` | List rules |
| `rookie permission allow` | Allow tool |
| `rookie permission deny` | Deny tool |
| `rookie permission ask` | Ask for tool |
| `rookie permission move` | Move rule |

## Agent Commands

| Command | Description |
|---------|-------------|
| `rookie agent list` | List agents |
| `rookie agent run` | Run agent |
| `rookie agent pipeline` | Run pipeline |
| `rookie agent parallel` | Run parallel |

## Skill Commands

| Command | Description |
|---------|-------------|
| `rookie skill list` | List skills |
| `rookie skill import` | Import skill |
| `rookie skill export` | Export skill |
