# Rookie Agent

AI-powered software engineering assistant with TypeScript orchestration and Rust compute engine.

## Features

- **Multi-Agent Orchestration** — Coder, Explorer, Reviewer, Architect, Planner, Evaluator
- **Long-Task Harness** — Session persistence, progress tracking, feature verification
- **Self-Improving Skills** — Auto-generate and optimize skills from usage
- **Structured Memory** — Episodic, semantic, and user model layers
- **Multi-Platform** — CLI, TUI, and gateway support (Feishu/Lark)
- **Built-in Tools** — 16+ tools for file, shell, git, web, and notebook operations

## Quick Start

```bash
npm install -g @rookie-agent/cli
rookie init --task "Fix the authentication bug"
rookie code
```
